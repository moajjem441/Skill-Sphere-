# 📚 SkillSphere – Online Learning Platform

## 🚀 Project Description

SkillSphere is a modern online learning platform where users can explore and enroll in various skill-based courses like Web Development, Design, Marketing, and more.  
It provides authentication, protected routes, course browsing, and a responsive user interface.

---

## 🌐 Live URL

👉https://skill-sphere-taupe.vercel.app

---

## 🎯 Purpose

The goal of this project is to build a full-stack style frontend learning platform using Next.js that allows users to:

- Register and login securely
- Use Google authentication
- Browse and search courses
- View course details (protected route)
- Manage user profile

---

## ✨ Key Features

- 🔐 Email & Google Authentication
- 📚 All Courses Page
- 🔍 Search Courses by Title
- 📖 Course Details Page (Protected)
- 👤 My Profile Page
- 🔥 Trending / New Releases Section
- 📱 Fully Responsive Design
- ⚡ Toast Notifications
- 🧭 Persistent Navbar & Footer
- ❌ Custom 404 Page
- ⏳ Loading UI Support
